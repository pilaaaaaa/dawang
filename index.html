数据存在你自己的 Supabase，不经过任何第三方服务器。
第一步：建表
登录 supabase.com → 新建项目 → 左侧 SQL Editor → 粘贴以下代码运行：

create table if not exists zuji_data ( id text primary key default 'main', payload jsonb not null, updated_at timestamptz default now() );

然后在 Project Settings → API 复制 Project URL 和 anon key。
